## 查看 

```sql
select auto_increment from information_schema.tables where table_schema='xxx_db' and table_name='xxx'; 
```

## 修改

```sql
alter table xxx_db.xxx auto_increment=1000;
```

> 坑：假设表中有ID大于你设置的值如1000，修改将不会生效
